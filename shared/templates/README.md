# AlvinAI Agency — Standard Document Templates

All agent deliverables MUST use the appropriate template from this folder.

## Templates

| Template | Used for | Who creates it |
|----------|----------|----------------|
| `campaign-brief.md` | New client campaigns | CMO / CEO |
| `ad-copy.md` | Facebook, Instagram, Google ads | Copywriter |
| `content-calendar.md` | 30-day social media plan | Content Strategist + Social Media Manager |
| `competitor-analysis.md` | Market research | Market Researcher |
| `influencer-brief.md` | Influencer partnership briefs | Social Media Manager |
| `monthly-client-report.md` | Monthly performance reports | Analytics Reporter |

## Rules

1. **Always start from the template.** Copy it, fill it out, never start from scratch.
2. **Remove all `<!-- INSTRUCTIONS -->` comments** before delivering to client or posting as a deliverable.
3. **CMO reviews all creative templates** (ad-copy, content-calendar, influencer-brief) before status changes to `in_review`.
4. **CEO checks template compliance** before marking any issue `in_review`.
5. Save your completed document as an issue attachment or document — not just a comment.

## Adding new templates

Create a new `.md` file in this folder and add it to the table above. Ping @CEO to update agent AGENTS.md files.
